DBMS: Beginners SQL Command Cheat Sheet

Database Management Systems is the software that handles the storage, retrieval, and updating of data in a computer system. Structured Query Language (SQL) is the standard language used for DBMS. This cheat-sheet summarizes frequently used commands for a single table

CREATING TABLES AND ADDING INFO TO THE TABLE

To see a summary of each command, click it.

CREATE TABLE celebs(id INTGER, name TEXT, age INTEGER);

CREATE TABLE will create new a table. In this example we create a new table called celebs with 'id', 'name', and 'age' as the columns

INSERT INTO celebs(id,name,age)VALUES(1,'Alex Mitzman', 26);

INSERT INTO will insert a new row into the table.

MAKING CHANGES TO THE TABLE

commands that will modify tables

ALTER TABLE celebs ADD COLUMN twitter_handle TEXT;

This adds a new column to the table. In this example 'celebs' is the table and 'twitter_handle' will be the column added to it.

DELETE FROM celebs WHERE twitter_handle IS NULL;

This will delete one or more rows. In this example it will delete all rows from the 'celebs' table that don't have a twitter_handle.

UPDATE celebs SET age = 30 WHERE id =1;

UPDATE will edit a row in the table. In this example it will change the age to 30 for id 1 in the table 'celebs'.

RETRIEVE DATA

How to return data from a table

SELECT name FROM celebs;

This will select all of the 'names' column for each row from the table 'celebs'

SELECT DISTINCT genre FROM movies;

Used to return a unique value. It filters all duplicate values. In the example, it will return all of the genres once from the 'movies' table

CONDITIONAL OPERATORS

Operators used to find more specific data to return

SELECT * FROM movies WHERE name BETWEEN 'A' AND 'J';

Used to filter within a certain range. In this instance it will return all of the rows data where the names are between 'A' and 'J'

SELECT * FROM movies WHERE name LIKE 'Se_en';

Useful operator when you want to compare similar values. In this example it returned all of the rows data where name is like 'Se_en'('_' is a wildcard character). It would return the rows for 'Seven' and 'Se7en'

SELECT * FROM movies WHERE rating > 8;

Equal(=), not equal(!=), greater than(>), less than(<), greater than or equal to(>=), and less than or equal to.(<=)

HOW TO ORDER DATA

Used to sort the data in ascending or descending order

SELECT * FROM fake_apps WHERE downloads > 2000 ORDER BY price ASC;

This will return every field in which the downloads are greater than 2000 and it will be sorted by price in ascending order.

LIST OF AGGREGATE FUNCTIONS

SQL aggregate functions return a single value, calculated from values in a column

SELECT column_name, AGGREGATE FUNCTION(column_name) FROM table_name;

COUNT()

Returns a number of rows

AVG()

Returns the average value

MAX()

Returns the largest value

MIN()

Returns the smallest value

SUM()

Returns the sum

ROUND()

Rounds a numeric field to a specified number of decimals

GROUPING AGGREGATE FUNCTIONS

The GROUP BY command which can only be used with aggregate functions

SELECT price, COUNT(*) FROM fake_apps WHERE downloads > 2000 GROUP BY price;

This will return a count of each price where the download value is greater than 2000

WILDCARDS

Powerful symbols that can represent many characters.

SELECT * FROM celebs

* Wildcard that allows you to select every column in a table

SELECT * FROM movies WHERE name LIKE 'Se_en';

'_' allows it to be any character.

SELECT * FROM movies WHERE name LIKE '%man%';

A% - matches that begin with 'A'
%a - will that match that end with 'a'
%man% - will match anything that contains 'man'
